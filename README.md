## Dataset Overview

The Iris dataset is a classic benchmark dataset in machine learning used for classification tasks.

- Source: UCI Machine Learning Repository
- Total Samples: 150
- Features: 4 numerical attributes
- Problem Type: Multiclass Classification
- Target Classes: Setosa, Versicolor, Virginica

## Features
- Sepal length (cm)
- Sepal width (cm)
- Petal length (cm)
- Petal width (cm)

## Objective
To classify iris flower species based on physical measurements using supervised learning techniques.

## Key Characteristics
- Balanced dataset (50 samples per class)
- No missing values
- One class is linearly separable (Setosa)
- Other two classes overlap partially
